## Go CMD USE

1. **bug** — 报告漏洞
2. **build** — 编译包和依赖
3. **clean** — 清除对象文件
4. **doc** — 展示源代码中的文档
5. **env** — 查看 Go 环境变量信息
6. **fix** — 修复因 Go 版本变化而导致的 API 兼容问题
7. **fmt** — 源代码格式化
8. **generate** — 代码生成
9. **get** — 添加依赖
10. **install** — 安装并编译包
11. **list** — 包/模块列表命令
12. **mod** — 模块维护命令
13. **work** — 工作区维护命令
14. **run** — 编译并运行
15. **test** — 测试
16. **tool** — 运行指定的 go 工具
17. **version** — 展示 Go 的版本信息
18. **vet** — 扫描并输出源代码中可能存在的问题
