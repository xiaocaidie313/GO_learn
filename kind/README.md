# kind

## 要点

- **类型定义** `type A B`：新类型，与原类型不兼容，可绑方法
- **类型别名** `type A = B`：等价于 B，不能绑独有方法
- **类型断言** `v, ok := x.(T)` 判断 interface 动态类型

## 对比

| | `type TwoDMap map[...]` | `type TwoDMap2 = map[...]` |
|---|------------------------|---------------------------|
| 新类型 | 是 | 否（别名） |
| 可绑方法 | 是 | 否 |
| 与原类型互换 | 需转换 | 直接互换 |
